# EF

This responsive template harnesses the power of the following libraries and resources:

* [Chart.js][1]
* [Google Maps][2]
* [Isotope][3]
* [Swiper][4]
* [ImagesLoaded][5]

The jQuery library is not used. Only pure JavaScript.

Browser support:

* IE9+
* Chrome+
* Firefox+
* Safari+
* Opera+
